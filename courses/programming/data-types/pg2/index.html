Data Types




  1. Derived Datatypes :

    These datatypes are derived from the primitve ones (predefined datatypes) and those are as follows:

    1. Functions
    2. Arrays
    3. Pointers
    4. References

    1. Functions:
      They are blocks of user defined code that are made for a specific task. They can be called by the programmer whenever needed , so as to execute the statements and hence prevents code rewriting.

      There’s one default function every C program must have: main()

      Syntax:

      returnType <functName>( functParameters)
      {
      //FunctionStatements
      }

      e.g.
      int decreaseBy5(int userVal)
      {
      return userVal - 5 ;
      }
      // This function does nothing but returns the value by subtracting "5" from the passed parameter.


    2. Array
      An array is a collection of similar type of data items , which are stored in contiguous locations in the memory. The array variable is used to represent a set of values.

      Syntax:

      dataType <name> [size];


    3. Pointers
      Pointers are datatypes that point to memory addresses. They can reference and dereference values and are used to implement dynamic datastructures.

      Syntax:

      dataType *<
      pointer_name>


    4. Reference
      References are aliases for existing variables. Variables can be referenced by putting ‘&’ in the declaration.


  2. User Defined DataTypes:

    1. Class
    2. Structure
    3. Union
    4. Enumeration
    5. TypeDef


    1. Class:
      These are the building blocks of Object Oriented Programming. They consist of data members and member functions, which are accessible via instances of class known as objects.
      Classes are blueprints for objects.

      Syntax:

      class <className>
      {
      accessSpecifier;
      dataMembers;
      memberFunctions( )
      {
      //Function Definition
      }
      }

    2. Structure:
      Structures are again user defined data types in C/C++. They are used to group data item into one single structure.

      Syntax:

      struct userDetails
      {
      char name[50];
      char address[100];
      int mob_number;
      };

    3. Union :
      Unions are user defined data types where all the consisting members share the same memory locations.
      eg.
      union Example
      {
      int x, y;
      };
      int main()
      {
      // A union variable e
      union Example e;
      // e.y also gets value 2
      e.x = 2;
      cout << "After making x = 2:"<< endl << "x = " << e.x << ", y = " << e.y << endl;
      // e.x is also updated to 10
      e.y = 10;
      cout << "After making Y = 10:"<< endl << "x = " << e.x << ", y = " << e.y << endl;
      return 0;
      }

    4. Enumeration:
      Enumerations assign names to integral constants , thus help to maintain readability of the program.

      Syntax:

      enum Result {Pass=1, Fail=0}

    5. Typedef:
      Its a keyword used to assign alternative names to existing datatypes. Its employed for user defined datatypes when the names get a little complex.

      Syntax :

      typedef <type> <name>
