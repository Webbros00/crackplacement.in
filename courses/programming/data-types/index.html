Data Types




DATATYPES in any programming language give us the base definition for declaring any variables or functions we wish to use in our program. The datatype also tells the compiler the size of the variable in prospect.

DataTypes can be mainly of two types:

  1. Primary Datatypes
  2. Derived Datatypes
  3. UserDefined Datatypes

    1. Primary Datatypes are the basic 5 types namely:

      1. Integer
      2. Float
      3. Character
      4. Boolean
      5. Void

      1. INTEGER:
        Integer dataypes are used to store integer values(whole numbers). They are defined using the int keyword.

        Type Size Range
        int or signed int 2 -32768 to 32767
        unsigned int 2 0 to 65,535
        short int or signed short int 1 -128 to 127
        unsigned short int 1 0 to 255
        long int or signed long int 4 -2,147,483,648 to 2,147,483,647
        unsigned long int 4 0 to 4,294,967,295

      2. FLOAT:
        Float can store all of real numbers (numbers with decimal points). They are defined using the float keyword.

        Type Size Range
        float 4 3.4E-38 to 3.4E+38
        double 8 1.7E-308 to 1.7E+308
        long double 10 3.4E-4932 to 1.1E+4932

      3. CHARACTER:
        Character dataype is used to store characters as values.
